# 淡白影视安卓端

[下载地址](https://netcut.cn/api/file/download/?note_id=a35c9a5e57e7342b&file_id=a35c9a5e57e7342b-1596105726184jn)

## 功能特性

+ 免VIP在线观看影视(资源来源网络)
+ 跨平台同步观看历史记录
+ 跨平台同步观看进度
+ p2p视频加速
+ 一起看观看,语音连麦
+ 倍速播放
